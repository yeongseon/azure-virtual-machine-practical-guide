# Azure Virtual Machine 실무 가이드

[English](README.md) | [한국어](README.ko.md) | [日本語](README.ja.md) | [简体中文](README.zh-CN.md)

MS Learn 문서를 기반으로 한 Azure VM 플랫폼 내부, 운영 및 트러블슈팅을 다루는 실무 가이드입니다.

## 범위

- ✅ 포함: Azure VM 플랫폼 개념, 운영, 보안, 모니터링, 트러블슈팅
- ❌ 제외: OS별 또는 애플리케이션 설치 튜토리얼

## 주요 내용

| 섹션 | 목적 |
|---------|---------|
| 시작하기 (Start Here) | VM 개요, 다른 컴퓨팅 옵션과의 비교, 학습 경로 |
| 플랫폼 (Platform) | Azure VM 작동 원리, 구성 요소, 수명 주기 |
| 베스트 프랙티스 (Best Practices) | 운영 환경에 적합한 설계 및 운영 가이드라인 |
| 운영 (Operations) | 단계별 운영 절차 및 구성 |
| 트러블슈팅 (Troubleshooting) | 증상 기반 진단 및 해결 |
| 참조 (Reference) | 빠른 조회 요약 테이블 |

## 콘텐츠 소스

모든 콘텐츠는 공식 [Microsoft Learn](https://learn.microsoft.com/en-us/azure/virtual-machines/) 문서를 기반으로 합니다.

## 로컬 빌드

```bash
pip install mkdocs-material pymdown-extensions
mkdocs build --strict
mkdocs serve
```
